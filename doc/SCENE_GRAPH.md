- viewport: devicePixel, view, height, ..
- blend mode
- render to texture
- shader variables: textures, uniforms
- gpu program
- draw sprites: type (program), vo pool

- blend mode

- vo pool (vertex data arrays)
- draw primitives: gl-primitives, draw elements
